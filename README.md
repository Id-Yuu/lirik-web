# lirik-web

Membuat Aplikasi Web dengan ES6, Custom Element, NPM, Module Bundler, dan AJAX

#### 1. Install with npm

```
npm install
```

#### 2. Run local

```
npm run dev
```

#### 3. Build Production

```
npm run build
```

## Feature

- Search lyrics
- Show lyrics

## Tech Stack

- Webpack
- axios
- javascript
- api lyrics
